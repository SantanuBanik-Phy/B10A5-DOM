What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way for web browsers to understand and work with HTML documents,the DOM is like a bridge between a webpage's content and JavaScript, enabling dynamic and interactive web pages.

How do you select an element from the DOM?

There are many ways to select an element from DOM,these are:

  • 1.getElementById()
  • 2.querySelector()
  • 3.getElementsByTagName()
  • 4.getElementsByClassName()
  • 5.querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where attach a single event listener to a parent element to handle events on its child elements, instead of adding multiple listeners to each child. It works due to event bubbling, where events propagate up the DOM tree. This method is useful because it improves performance by reducing the number of event listeners and simplifies handling dynamically added elements. It's efficient for managing events in large or dynamic DOM structures.

How do you manipulate an element's attribute and styles using the DOM?

There are many ways to manipulate an element's attributes and styles using the DOM:

  • 1.setAttribute()
  • 2.getAttribute()
  • 3.removeAttribute()
